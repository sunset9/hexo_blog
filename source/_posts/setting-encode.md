웹에서는 EUC-KR 이나 UTF-8을 많이 사용한다.

Window - Preferences 선택 후
  1. General 항목
	\- Workspace 항목
	\- 하단의 Text file encoding의
	&nbsp;&nbsp;&nbsp;&nbsp;Other: UTF-8 설정

  2. Web 항목
	\- CSS Files
	\- HTML Files
	\- JSP FIles
	세가지 항목 모두	ISO 10646/Unicode(UTF-8) 로 설정

위와같이 1,2 설정을 해준다.
